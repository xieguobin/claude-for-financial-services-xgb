**业绩预览**

描述：通过预测模型、情景框架和关键观察指标构建业绩前瞻分析。适用于公司季度业绩发布前，用于准备持仓备忘、设定多空情景，并确定推动股价变动的因素。触发词包括“业绩预览”、“［公司］业绩该关注什么”、“业绩前瞻”、“业绩布局”或“预览［公司］的Q［X］业绩”。

## 工作流程

### 步骤1：收集背景信息

- 确定公司及报告季度
- 通过网络搜索获取一致预期（营收、每股收益、关键细分业务指标）
- 查找业绩发布日期和时间（盘前还是盘后）
- 回顾公司上一季度业绩电话会议中的任何指引或点评

### 步骤2：关键指标框架

建立针对该公司特定的“关注什么”框架：

**财务指标：**
- 营收 vs. 一致预期（总额及各细分业务）
- 每股收益 vs. 一致预期
- 利润率（毛利率、营业利润率、净利率）—— 扩大还是收窄？
- 自由现金流
- 未来指引 vs. 一致预期

**运营指标**（因行业而异）：
- 科技/软件即服务：年度经常性收入、净留存率、剩余履约义务、客户数量
- 零售：同店销售额、客流量、客单价
- 工业：在手订单、订单出货比、价格 vs. 销量
- 金融：净息差、信贷质量、贷款增长、手续费收入
- 医疗健康：处方量、患者量、管线进展

### 步骤3：情景分析

构建3种情景及其对应的股价影响：

| 情景 | 营收 | 每股收益 | 关键驱动因素 | 股价反应 |
|------|------|----------|--------------|----------|
| 乐观 |      |          |              |          |
| 基准 |      |          |              |          |
| 悲观 |      |          |              |          |

针对每种情景：
- 从经营层面需要发生什么
- 管理层点评会传递什么信号
- 历史背景——该股在类似业绩发布后如何变动

### 步骤4：催化剂核对清单

识别决定股价反应的3—5个关键事项：

1. ［指标］ vs. ［一致预期/市场传闻数字］—— 为何重要
2. ［指引项目］—— 买方期望听到什么
3. ［叙事变化］—— 任何战略变动、并购、重组

### 步骤5：输出

一页纸的业绩预览，包含：
- 公司、季度、业绩发布日期
- 一致预期汇总表
- 关键观察指标（按重要性排序）
- 乐观/基准/悲观情景表
- 催化剂核对清单
- 交易布局：近期股价表现、期权隐含波动区间

## 重要说明

- 一致预期会变动——务必注明预期数据来源及日期
- 买方调研得出的“市场传闻数字”通常比公开的一致预期更具参考价值
- 历史业绩后股价反应有助于校准预期（搜索“［公司］业绩后股价反应历史”）
- 期权隐含波动区间反映市场预期——将其与你的情景比较
